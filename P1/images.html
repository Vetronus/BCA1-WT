FONT STYLE

Font style refers to the size,weight,color and style of typed characters within a document in an email or a webpage.In other words,the font style changes the appearence of a complete set of characters that make up a typeface or face Font style.

SIZE ATTRIBUTE

The size attribute defines the width of the input and the height of the selected element.For the input,if the type attribute flare is text or password then it's the number of characters.

WEIGHT ATTRIBUTE

The font-weight property lets how thick or thin characters in text should be displayed.Default value:normal.

COLOR ATTRIBUTE

The HTML color Attribute is used to specify the text color inside the lement.

STYLE ATTRIBUTE

The Style attribute specifies an inline style for an element.The style attribute will override any style set globally.

IMAGE TAG

The tag is used to embed an image in an HTML page.Images are not technically inserted into a web page,images are linked to web pages.The tag creates a holding space for the influenced image
